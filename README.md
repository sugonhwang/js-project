### 1. UP & Down Game

- **데모**: [Live Demo](https://sghwang-jsproject.netlify.app/js-updown-game/)
- **설명**: 임의의 숫자를 맞추는 게임
- **기술 스택**: HTML, CSS, JavaScript
- **주요 기능**:

  - 숫자 중복 입력 방지
  - 게임 재시작
  - 게임 종료 시 정답 공개(예정)
  - 제한 시간(예정)
    - 숫자 하나 입력할 때마다 제한 시간 리셋
